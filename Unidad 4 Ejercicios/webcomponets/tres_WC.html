slot

Un marcador de posicion dentro de un componente web que los usuarios pueden rellenar con su propio marcador, con el efecto de componer diferentes alboles DOM juntos

Name

El atributo name del slot
Template

Un mecanismo para guardar contenido en el lado del cliente que no se renderiza cuando se carga sino que posteriormente se puede instanciar en tiempo de ejecucion usando javaScript
